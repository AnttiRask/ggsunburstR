# Plot a sunburst_data object

Dispatches to
[`sunburst()`](https://anttirask.github.io/ggsunburstR/reference/sunburst.md).
Until
[`sunburst()`](https://anttirask.github.io/ggsunburstR/reference/sunburst.md)
is implemented, this prints an informational message.

## Usage

``` r
# S3 method for class 'sunburst_data'
plot(x, ...)
```

## Arguments

- x:

  A `sunburst_data` object.

- ...:

  Passed to
  [`sunburst()`](https://anttirask.github.io/ggsunburstR/reference/sunburst.md).

## Value

A ggplot object (once
[`sunburst()`](https://anttirask.github.io/ggsunburstR/reference/sunburst.md)
is available), or `x` invisibly.
